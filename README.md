# FitScore 360

An AI-powered career companion that analyzes resume-to-job fit, providing quantitative scoring, skill gap analysis, and personalized roadmap generation aligned with the Handshake ecosystem.

## Features

- **Resume Analysis**: Upload your resume and get instant feedback.
- **Job Fit Scoring**: Quantitative scoring based on job descriptions.
- **Skill Gap Analysis**: Identify what skills you need to land your dream job.
- **Personalized Roadmap**: Step-by-step guide to bridge the gap.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **AI**: Google Gemini API
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```
2. Navigate to the project directory:
   ```bash
   cd YOUR_REPO_NAME
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

To start the development server:
```bash
npm run dev
```

## Environment Variables

This project uses the Gemini API. Ensure you have your API key configured if running locally.

```env
GEMINI_API_KEY=your_api_key_here
```
